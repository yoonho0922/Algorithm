## DFS와 BFS

#### 그래프 input 방식

* 연결 리스트
  * 시간적 측면에서 유리 (edge가 적을 경우)
  * 2차원 배열 또는 vector 배열로 구현
* 행렬
  * 2차원 배열로 구현

#### DFS

* 스택을 vector로 구현
* 재귀함수로 구현
* visit 배열이 있어야함

#### BFS

* queue로 구현
* visit대신 dist로 구현 가능
* 문제에서 **최소 경로**를 요구할 때 자주 쓰임
